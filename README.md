# 🤖 ОтмазкоГенератор 3000

Это гибридная интеллектуальная система (веб-приложение на FastAPI + Telegram-бот на Telebot), созданная для генерации правдоподобных или абсурдно-смешных оправданий с использованием искусственного интеллекта Llama 3.1 (через Groq API).

---

## 🛠 Технологический стек

* **Бэкенд:** Python, FastAPI, Uvicorn
* **Фронтенд:** HTML5, CSS3 (Glassmorphism UI), JavaScript (AJAX / Fetch API)
* **Telegram Бот:** pyTelegramBotAPI (Telebot), Threading (многопоточность)
* **База данных:** Локальная СУБД на базе одного JSON-файла (`db.json`)
* **ИИ Интеграция:** Groq API, модель `llama-3.1-8b-instant`
* **Безопасность:** python-dotenv (переменные окружения)

---

## 🌟 Ключевые возможности системы

1. **Гибридная синхронизация:** Веб-приложение и Telegram-бот подключены к одной общей базе данных `db.json`. Все отмазки, сгенерированные на сайте или в боте, мгновенно отображаются в истории на главной странице веб-интерфейса.
2. **Два режима генерации ИИ:** 
   * 🛡️ *Правдоподобный (серьезный):* Вежливые, приземленные и реалистичные оправдания, в которые поверит любой преподаватель.
   * 🤪 *Абсурдный (смешной):* Фантастические, глупые и дикие истории для поднятия настроения.
3. **Динамическая настройка (Промпт-инжиниринг):** Через команду бота `/prompt` пользователь может полностью переписать характер ИИ прямо «на лету».
4. **Современный веб-интерфейс:** Темная тема с неоновыми акцентами, интерактивные кнопки выбора режимов и анимированный спиннер во время генерации ответа.
5. **Безопасность данных:** Все токены и API-ключи скрыты в конфигурационном файле `.env` и защищены от утечки в публичный доступ через `.gitignore`.
